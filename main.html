IntelliJ IDEA 단축키 설명하기

단축키 설명
Alt + Enter 코드에서 해당 오류 위에 커서를 놓고 Alt + Enter를 누르면 문제에 대한 수정 제안 목록이 표시되고 선택하면 코드를 수정해줍니다.
F2 제안 탐색 시 F2를 활용해 마우스 사용 없이 다음 오류, 경고, 제안으로 이동할 수 있습니다.
Alt + 1 마우스 사용 없이 도구(프로젝트) 창을 열 수 있습니다.
Esc 프로젝트 창에서 Esc를 눌러 에디터로 돌아올 수 있고, 변경 사항을 적용하지 않고 모든 팝업을 닫는데 유용합니다.
Ctrl + E Ctrl + E 를 사용하면 가장 최근 열었던 파일 대화상자를 열 수 있습니다.
Ctrl + B Ctrl + B 를 누르면 심볼의 선언으로 이동할 수 있습니다.
Alt + F7 Alt + F7 를 누르면 모든 사용 위치가 표시됩니다.
Tap Ctrl Twice Ctrl을 두면 누르면 Run Anything 창이 열립니다.
Ctrl + W, Ctrl + Shift + W Ctrl + W, Ctrl + Shift + W를 사용해 커서 주변의 코드 섹션을 확장하거나 축소할 수 있습니다.
Ctrl + / 주석을 간편하게 추가할 수 있습니다.
Shift + Ctrl + Enter 구문을 자동 완성시켜줍니다.
Ctrl + Alt + L 간편하게 현재 파일의 서식을 프로젝트 표준에 맞춰줍니다.
Shift + Ctrl + Alt + T 모든 리팩토링을 표시해 줍니다.
Shift + Ctrl + A 액션 검색
Tap Shift Twice 무엇이든 검색 가능한 검색 창을 표시해 줍니다.
